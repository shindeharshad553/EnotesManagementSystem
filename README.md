# EnotesManagementSystem
The E-Notes Management System is a web-based application designed to help users organize, manage, and store their notes efficiently. Developed using JSP, Servlets, and JDBC, this project provides users with the ability to create, edit, delete, and view their notes online.

Features
1) User Authentication:
   User Registration with validation.
   Secure Login and Logout functionality.
       
2) Notes Management:
  Create new notes.
  View all saved notes in a user-friendly dashboard.
  Edit existing notes.
  Delete notes.

3) Session Management:
  Persistent user sessions to enhance security and usability.

4) Error Handling:
  Proper error messages for invalid actions or database issues.

5) Database Integration:
  Efficient storage and retrieval using JDBC with a MySQL database.
